# AI Career Guidance System

यह एक वेब-आधारित AI करियर गाइडेंस सिस्टम है जो छात्रों को उनकी शिक्षा, स्ट्रीम और बजट के आधार पर व्यक्तिगत करियर सुझाव प्रदान करता है।

---

## ## मुख्य विशेषताएँ (Features)

* **व्यक्तिगत सुझाव:** यूजर की शिक्षा (10वीं/12वीं), स्ट्रीम (PCM, PCB, आदि), और फीस के बजट के आधार पर सुझाव।
* **स्मार्ट रैंकिंग:** AI स्कोरिंग सिस्टम जो भविष्य के स्कोप और फीस के आधार पर कोर्स को रैंक करता है।
* **व्यापक जानकारी:** कोर्स, संबंधित प्रवेश परीक्षाओं और भविष्य की नौकरियों के लिए सुझाव प्रदान करता है।
* **प्रोफेशनल UI:** एक साफ, आधुनिक और उपयोग में आसान इंटरफ़ेस।
* **डेटाबेस संचालित:** सभी डेटा को CSV फाइलों से एक मजबूत SQLite डेटाबेस में संग्रहीत किया गया है।

---

## ## टेक्नोलॉजी स्टैक (Tech Stack)

* **बैकएंड:** Python, Flask
* **डेटा हैंडलिंग:** Pandas, SQLite
* **फ्रंटएंड:** HTML, CSS
* **डिप्लॉयमेंट:** PythonAnywhere

---

## ## इस प्रोजेक्ट को स्थानीय रूप से कैसे चलाएं (How to Run Locally)

1.  **रिपॉजिटरी क्लोन करें:**
    ```bash
    git clone <your-repo-link>
    cd career-guidance-system
    ```

2.  **एक वर्चुअल एनवायरनमेंट बनाएं और सक्रिय करें:**
    ```bash
    python -m venv venv
    source venv/bin/activate  # macOS/Linux
    venv\Scripts\activate    # Windows
    ```

3.  **आवश्यक लाइब्रेरी इंस्टॉल करें:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **डेटाबेस बनाएं:**
    (सुनिश्चित करें कि `data` फोल्डर में आपकी `.csv` फाइलें हैं)
    ```bash
    python import_data.py
    ```

5.  **एप्लीकेशन चलाएं:**
    ```bash
    python app.py
    ```
    अब अपने ब्राउज़र में `http://127.0.0.1:5000/` पर जाएं।
